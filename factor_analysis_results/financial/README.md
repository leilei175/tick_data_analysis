# 财务指标因子分析说明

## 分析概述

- **股票池**: 中证1000成分股 (1000只)
- **数据范围**: 2025年1月1日 - 2025年12月31日
- **分析因子**: 18个财务指标

## 因子有效性评价标准

| IC IR 范围 | 评价 | 含义 |
|-----------|------|------|
| IR > 0.5 | 强有效 | 因子预测能力强 |
| 0.3 < IR ≤ 0.5 | 有效 | 因子有较好预测能力 |
| 0.1 < IR ≤ 0.3 | 弱有效 | 因子有一定预测能力 |
| 0 < IR ≤ 0.1 | 中性 | 因子预测能力很弱 |
| IR ≤ 0 | 无效 | 因子无预测能力 |

## 关键发现

### TOP 5 最佳因子
1. **current_ratio** (流动比率) - IC IR = 0.103
2. **cf_to_nincome** (现金流/净利润) - IC IR = 0.085
3. **gross_margin** (毛利率) - IC IR = 0.079
4. **op_margin** (营业利润率) - IC IR = 0.079
5. **net_margin** (净利率) - IC IR = 0.078

### 不推荐因子
1. **profit_growth** (利润增长率) - IC IR = -0.502 (负向)
2. **revenue_growth** (营收增长率) - IC IR = -0.415 (负向)
3. **debt_ratio** (资产负债率) - IC IR = -0.148 (负向)
4. **pb** (市净率) - IC IR = -0.104 (负向)

## 分层收益计算说明

### 分组方法
使用 **rank 分组** 方法，避免极端值导致的分组不均匀问题：
- 每日对所有股票的因子值进行排名
- 按排名将股票分成5组 (Q1-Q5)
- Q1 = 因子值最低的20%
- Q5 = 因子值最高的20%

### 过滤规则
- 排除涨停股票 (pct_chg >= 9.5%)
- 排除跌停股票 (pct_chg <= -9.5%)
- 排除成交不足的股票 (amount < 100万)

## 投资建议

### 推荐使用的因子
1. **current_ratio** - 流动比率高的公司未来收益更好
2. **盈利能力指标** (gross_margin, roa, roe) - 高盈利能力公司有正超额收益
3. **cf_to_revenue** - 现金流健康的公司表现更好

### 不推荐使用的因子
1. **成长因子** (revenue_growth, profit_growth) - 增长率与收益呈负相关（均值回归）
2. **估值因子** (pb, ps) - 在中证1000中无明显预测能力
3. **高负债** (debt_ratio) - 高杠杆公司收益更差
